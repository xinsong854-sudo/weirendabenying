# 隐私说明

本网站不在后端持久化保存以下信息：

- Neta Token
- 手机号
- 验证码
- IP 地址
- User-Agent
- 设备指纹
- 地理位置

## Token

Neta 登录 Token 仅保存在用户浏览器本地 `localStorage` 中，用于本地登录状态恢复。

请求后端接口时，Token 会通过 `x-token` 请求头临时传递，用于本次请求身份验证或代理 Neta API。后端不将 Token 写入数据库、不写入日志、不落盘。

## 手机号和验证码

手机号和验证码仅用于登录时代理调用 Neta / Talesofai 登录接口。

本网站后端不保存手机号和验证码。

## 后端保存的数据

后端只保存用户主动同步的站内内容：

- 用户 UUID
- 昵称
- 头像 URL
- 角色权限
- 头像框
- 签名
- 论坛发言
- 评论
- Wiki 投稿
- 身份卡 / CoC 车卡
- 私聊内容

## 限流

限流只使用内存中的临时时间戳。登录用户按 Token 解析出的用户 UUID 分桶，匿名请求进入匿名桶。不记录 IP，不保存 Token。
