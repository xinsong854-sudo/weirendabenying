# 安全说明

## 前端安全

- 正式前端由 `/pages/vue-app/src` 构建到 `/pages/dist`。
- 项目不使用 `v-html` 渲染用户输入。
- 用户文本通过 Vue 插值渲染，默认会 HTML escape。
- 图片 URL 通过 `safeUrl()` 校验，只允许 `http://` 和 `https://`。
- 前端 API 封装带超时控制，避免请求长期挂起。

## 后端安全

- 后端不保存 Token、手机号、验证码、IP、User-Agent、设备指纹或地理位置。
- 请求体最大 1MB。
- JSON 解析失败返回 400，不抛出未处理异常。
- 用户输入经过 `clean_text()` 控制长度并移除控制字符。
- 外链图片经过 `safe_public_url()` 校验。
- 数据库操作使用参数化 SQL。
- 轻量限流仅使用内存中的临时计数，不写入数据库。

## CSP

当前 CSP 采用保守策略，保留验证码所需的外部脚本兼容性。不要在未完整验证极验验证码域名前贸然移除 `unsafe-inline` / `unsafe-eval`，避免再次导致验证码无法登录。

## 不做强反调试

本项目不使用无限 `debugger`、控制台检测死循环、强制关闭 DevTools 等反调试逻辑。这类逻辑容易破坏验证码、移动浏览器和正常用户体验，也可能被安全扫描判定为可疑行为。
