# 奇遇剧本范例

以下是不同类型的完整字段范例。这些不是可以直接复制的模板，而是可以参照的标杆——展示「卓越剧本」的实际形态。

---

## 范例一：历史穿越——汴京最后的三天

**类型**：历史穿越 / 政治悬疑
**基调**：历史正剧质感；沉重、真实、命运感

```
name: "汴京最后三天"

subtitle: "你知道靖康之难会发生。你不知道的是，一个不该出现的人，正在改变它的经过。"

status: PUBLISHED

mission_plot: |
  公元1127年，靖康元年十一月。
  金兵已兵临汴京城下。徽宗内禅，钦宗登基，朝廷乱作一团。
  史书上写得清楚：三天后，城破。

  你穿越到一名汴京城内太学生的身上。这具身体的原主人三天前
  刚刚死于一场街头斗殴——没有人注意到换了灵魂。你有他的记
  忆、他的人脉、他的欠债，和他的一个未完成的约定。

  你知道历史的走向。你知道城会破，皇帝会被掳，山河会变色。
  但你不知道的是：昨天，一名不在史书上出现的「使者」出现在
  徽宗的密室中，带来了一封不知来源的密函。这不在任何你读过
  的史料里。

  有人比你先一步来了这里。他们的目的，你还不知道。

mission_task: |
  在城破之前（三天内），查明密函的来源和使者的真实身份。
  你无法阻止靖康之难——历史不会因你而改变。但你或许能
  弄清楚「那个改变了某些细节的人」究竟做了什么，以及
  他的行为对历史造成了什么本不该有的偏移。
  次要目标：完成原身的那个约定；在城破前离开汴京。

mission_plot_attention: |
  历史准确性：徽宗、钦宗、李纲等真实历史人物的性格和立场遵循
  史料记载。徽宗懦弱但有审美；钦宗优柔寡断；李纲主战且孤立。
  不得将历史人物随意美化或丑化。

  语言风格：保持宋代书面语感，NPC 对话有文言色彩但不晦涩难懂。
  不使用现代网络用语（「YYDS」「绝绝子」等）。

  无法改变的历史：玩家的行动在历史的缝隙中展开。城破是确定
  事件，不可阻止。玩家能影响的是「缝隙内的细节」，而非大势。
  不要让玩家产生「我能拯救北宋」的错误期待。

  谜团不主动揭示：使者的身份和密函的内容必须通过调查获得，
  不会被 NPC 主动告知。每条线索都指向新的问题，而非直接答案。

  内容边界：政治暴力存在但以后果呈现，不描述过程。无性内容。
  这是悲剧叙事——不强制乐观结局。
```

**为什么有效**：
- 开场立即建立「已知走向 × 未知变量」的核心张力
- `subtitle` 把玩家的历史知识本身变成悬念来源
- `mission_plot_attention` 明确了历史准确性要求、语言风格、以及最重要的——「不要让玩家产生拯救北宋的幻觉」
- `mission_task` 目标可实现但不轻松，且与历史宿命感并存

## 范例二：末世生存——长沙隔离区第七天

**类型**：末世生存 / 人性考验
**基调**：资源稀缺；道德代价；没有标准答案

```
name: "隔离区·第七天"

subtitle: "解药的消息传来了。只有一个剂量。你们有九个人。"

status: PUBLISHED

mission_plot: |
  T 病毒从武汉扩散到全国，长沙在第三天进入军事封锁。
  你是一名地铁站维护工，封锁前最后一班车将你和其他乘客
  困在了芙蓉广场站地下。

  现在是第七天。
  地面广播三天没有更新了。手机信号断断续续，偶尔能刷到
  末世前最后几条朋友圈——日期停在了四天前。
  同行的九个人里，有两个开始发烧。

  昨晚有人偷了食物储备里的最后三袋方便面。没人承认。

  今天早上，有人的对讲机接收到一段信号：「芙蓉广场方向
  有幸存者的，到五一广场站集合。我们有解药。数量有限。
  先到先得。」

  五一广场站距离这里步行二十分钟。地面的情况，你不知道。
  两个发烧的人，能否撑到那里，你不知道。
  发出信号的人，是否真的有解药，你不知道。

mission_task: |
  决定如何回应那个信号，并执行你的决定。
  无论你选择什么，都要在资源耗尽之前做出选择。
  九个人中，你能带走多少人，取决于你的每一个决定。
  没有「正确答案」——只有你愿意付出的代价。

mission_plot_attention: |
  道德代价是真实的：帮助两个发烧的人可能意味着放慢其他人。
  信任陌生人可能是生存的关键，也可能是陷阱。每一个善意的
  行动都有机会成本。叙事者不评判玩家的选择，但后果是真实的。

  没有无敌主角：玩家是一名普通维护工，没有特殊技能或战斗
  能力。他们面对的困境与其他幸存者完全相同。

  人物弧线：九个同行者各有性格和隐藏的动机。食物被盗的人
  仍在其中。发烧的两人之一知道一些她没有说出口的事。

  不强制乐观结局：好人可能死去，道德选择可能带来更差的后果。
  但每一个结局都必须感觉是「由玩家的真实选择导致的」，而不是随机的悲剧。

  内容边界：暴力存在但以后果呈现。无性内容。心理压力和道德
  困境是主要的叙事张力来源。
```

## 范例三：武侠江湖——问剑录

**类型**：武侠 / 恩怨录
**基调**：快意恩仇与道义困境并存；江湖不只是打架

```
name: "问剑录"

subtitle: "你师父死于一场被称为「意外」的比武。凶手现在是武林盟主。"

status: PUBLISHED

mission_plot: |
  三年前，天机门掌门凌云鹤死于与玄冥派宗主赵天煞的「切磋」
  之中。验尸的大夫说是心疾突发。在场的十七名见证人全部作证。
  一个月后，赵天煞被推举为新任武林盟主，无人异议。

  你是凌云鹤的关门弟子，当日在场的第十八个人。
  你看到了赵天煞出手的那一剑。
  你没有作证，因为没有人问你。

  三年后，你从隐居之地回到江湖。不为复仇——至少你告诉
  自己是这样。你师父有句话，生前说了许多次：「剑问是非，
  不问输赢。」

  但江湖问的从来不是是非。

  入城第一天，赵天煞的二弟子在茶馆里认出了你。

mission_task: |
  在江湖中查明三年前那剑的真相。
  「真相」可能不只是凶手是谁——也包括你师父为什么必须死，
  以及谁在背后让这件事发生。
  你如何对待这个真相，由你决定：公诸于众、私下了结、
  或者发现它比复仇更沉重。

mission_plot_attention: |
  语言风格：武侠江湖感。NPC 使用武林切口（「道上的朋友」
  「今日有缘」「承让」）。打斗以招式意境描述为主，而非伤害
  数值（「一剑如秋风掠草，快而无声」而非「造成50点伤害」）。

  赵天煞不是纯粹的反派：他有他的逻辑，他的手段或许是江湖
  中某种扭曲的秩序维护。叙事者不提前定性他的「恶」，让玩家
  自己判断。

  道义困境是核心：「证明真相」可能意味着引发武林大乱；
  「私下了结」可能意味着正义从未发生；「放下」可能意味着
  对师父的背叛。没有完美的选项。

  信息门控：三年前在场的十七名证人各有立场和秘密。真相
  需要从不同的叙述中拼凑。没有一个人知道全部，包括赵天煞。

  内容边界：武侠暴力是叙事的一部分，以意境和后果呈现。
  无性内容。道义冲突是主要张力，而非单纯的武力对抗。
```
