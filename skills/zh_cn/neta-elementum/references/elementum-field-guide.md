# 元素字段说明手册

`create_elementum` 和 `update_elementum` 所有字段的详细说明与填写规范。

---

## 必填字段

### `name`

元素名称，最多 128 字符。

- 建议简短、有辨识度
- 创建后通过 `/元素名` 在 `make_image` 中引用
- 示例：`RE4村庄`、`水墨风`、`武士刀`、`战斗站姿`

---

### `artifact_uuid`（create 必填）

元素代表图的素材 UUID，来自 `make_image` 返回的 `artifacts[0].uuid`。

- 代表图应准确体现元素所表示的视觉概念
- 建议选择视觉效果最清晰、最典型的那一张
- 此图同时作为元素的封面图展示

---

### `prompt`（create 必填）

图像模型直读的生图指令。

**两个关键属性：**

1. **可直用** — 可以直接粘贴到 `make_image --prompt` 中单独使用
2. **可组合** — 与其他角色引用、描述词组合时不产生冲突

**写法规范：**
- 聚焦核心视觉特征，避免无关描述
- 不含特定角色（除非是以某角色为锚的姿势类元素）
- 语言可以是中文（8_image_edit 模型支持中文），但建议清晰精准

**各类型示例：**

```
# 场景类
生化危机4风格欧洲中世纪废弃村庄，破旧石屋，燃烧篝火，浓雾弥漫，枯木，恐怖压抑氛围，写实风格

# 物品类
日本武士刀，细长刀身，紫色刀鞘，金色护手，精钢刃，金属反光，白色背景

# 风格类
水墨画风格，留白美学，山水意境，笔触自然，中国传统绘画

# 姿势类
女性角色，回眸站姿，长发飘动，侧身45度，优雅，白色背景，全身像

# 氛围类
赛博朋克霓虹光效，紫色蓝色粉色混合光晕，雨夜反光地面，科技感
```

---

## 可选字段

### `description`

供 Agent 读取的使用说明。**这是 Agent 理解该元素、决定何时及如何使用它的核心字段。**

**推荐格式：**
```
此元素表示[概念描述]。使用时[方法和搭配建议]。参考图展示[说明]。[注意事项（如有）]。
```

**各类型示例：**

```
# 场景类
此元素表示生化危机4标志性的废弃欧洲村庄场景，充满恐怖压抑氛围。
使用时搭配角色引用（@角色名）和夜晚、阴暗、恐怖等描述词，适合恐怖题材或游戏风格生图。
参考图为RE4游戏村庄场景的视觉风格复现。
注意：以环境为主，生图时若需要人物，请单独描述角色动作和位置。

# 风格类
此元素表示日式水墨画风格，强调留白美学与山水意境。
使用时可搭配任意场景或角色，适合诗意、古典、东方美学题材。
参考图为传统水墨风格测试图。可叠加使用多个场景描述。

# 梗图类
此元素表示"星球大战"系列中的经典梗——Padmé Amidala 指着右侧的表情包。
使用时描述人物指向方向和表情即可激活该梗的氛围。
参考图为原梗图风格的二次元复现。适合轻松幽默场景。
```

> **重要：** description 质量直接影响 Agent 在后续创作中对该元素的正确使用。建议尽量写清楚"何时用"和"怎么用"。

---

### `ref_image_uuid`

概念锚定参考图的素材 UUID，用于视觉风格锚定（可选）。

**适合场景：**
- 有明确参考原图（如游戏截图、电影截图）
- 需要图像模型精准复现特定视觉风格
- 元素的视觉特征难以仅用文字描述

参考图来源：
- 通过 `make_image` 生成的图片的 `artifacts[0].uuid`
- 通过 `read_collection` 获取的图片 artifact

---

### `accessibility`

可见性设置。

| 值 | 含义 |
|----|------|
| `PUBLIC`（默认）| 所有用户可搜索和使用 |
| `PRIVATE` | 仅创建者可见和使用 |

---

## update_elementum 特有字段

### `tcp_uuid`（必填）

元素的唯一标识 UUID，通过 `request_character_or_elementum` 或 `create_elementum` 返回值获取。

---

## 字段关系总结

```
┌─────────────────────────────────────────────────────┐
│                 元素 Token (TCP/Elementum)           │
│                                                     │
│  给图像模型读                给 Agent 读              │
│  ──────────────             ─────────────────       │
│  prompt (生图指令)            description (使用说明)  │
│  ref_image_uuid (风格锚定)                            │
│                                                     │
│  anchor                                             │
│  ──────                                             │
│  artifact_uuid (代表图，封面展示)                      │
└─────────────────────────────────────────────────────┘
```

## prompt vs description 对比

| 维度 | prompt | description |
|------|--------|-------------|
| **读者** | 图像模型 | Agent / 用户 |
| **目的** | 直接生图 | 解释如何使用 |
| **语言** | 视觉标签，精准 | 自然语言，说明性 |
| **示例** | `赛博朋克霓虹，雨夜，紫色光晕` | `此元素表示赛博朋克夜景氛围，使用时搭配城市场景…` |
